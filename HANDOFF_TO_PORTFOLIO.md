# SmartBot — 作品集更新交接

## 项目概述

AI 智能客服微信小程序，帮助企业快速部署智能客服，用户扫码即可与 AI 对话。

## 展示信息

### 基本信息
- **项目名称**: SmartBot
- **类型**: 微信小程序
- **在线地址**: 体验版二维码（需邀请成为体验成员）

### 技术栈
- 微信原生小程序
- DeepSeek / OpenAI API 接入
- 流式输出 + 打字机效果
- Apple iOS 设计风格

### 功能亮点
- 🤖 AI 智能对话 - 基于大语言模型的智能问答
- ⚡ 极速响应 - 流式输出，毫秒级反馈
- 🔧 多服务商支持 - DeepSeek / OpenAI / 自定义 API
- 🎨 Apple 风格 UI - 简洁优雅的 iMessage 风格界面
